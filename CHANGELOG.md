# Vibe Prompt Architect — Changelog

This file is the authoritative version history for both SKILL.md (Claude Skill) and vibe-prompt-architect-knowledge.md (the Gemini Gem's content/knowledge file). The two files are versioned together — a single version number covers changes to either or both. The Gem router (vibe-prompt-architect-gem.md) is FILE-SPECIFIC and is not versioned here. SYNC-MANIFEST.md governs which locations to update on each change and defines the pre-commit process.

## Maintenance

When making any change to either file: (1) open SYNC-MANIFEST.md and identify the Feature Touch Map entries affected; (2) describe the change to Claude with all three files (SKILL.md, vibe-prompt-architect-knowledge.md, SYNC-MANIFEST.md) — Claude applies it consistently to both; (3) add a row to the Version History table above for every change, following semantic versioning; (4) update the **Current version** line above; (5) update the knowledge file's version marker line (*Version X.Y.Z · date*) directly under its H1; (6) update the SKILL.md YAML comment and italic byline to match; (7) update SYNC-MANIFEST.md Sync Health Log and any Feature Touch Map entries if the change adds a new feature or section.
